WeDeployEmail

public class WeDeployEmail : WeDeployService

Helper to communicate with Email service in WeDeploy. You can create a message and send it in a fluent way:

WeDeploy.email("auth-service-url")
    .authorize(auth: TokenAuth(token: "someToken"))
    .from("me@me.com")
    .to("you@you.com")
    .message("message")
    .send()
